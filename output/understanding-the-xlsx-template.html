5 Understanding the XLSX Template

The Excel file consists of multiple sheets that correspond to different database tables:

  1. Project & Site Data (Includes site_code, plot_code, profile_code, latitude, longitude)
  2. Soil Layer Data (Includes element_code, upper_depth, lower_depth, ph, soc, clay, etc.)
  3. Procedures Mapping (Links property_phys_chem_id with procedure_phys_chem_id and unit_of_measure_id)

These sheets are extracted and transformed before being injected into PostgreSQL.
